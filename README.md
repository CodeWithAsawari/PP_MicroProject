# 📘 Personal Study Organizer

A simple and colorful Python Tkinter project to manage study tasks and homework efficiently.

This application helps students:
- Add study tasks
- Mark tasks as completed
- Delete tasks
- Store tasks permanently using JSON file storage

---

# 🚀 Features

✅ Add Subject and Homework  
✅ Select Difficulty Level (Easy / Medium / Hard)  
✅ Mark Tasks as Completed  
✅ Delete Tasks  
✅ Scrollable Task List  
✅ Permanent Data Storage using JSON  
✅ User-Friendly GUI using Tkinter  

---

# 🛠 Technologies Used

- Python
- Tkinter
- JSON
- Object Oriented Programming (Class & Objects)
